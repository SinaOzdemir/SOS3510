Chapter 2 Setting up R and R studio

After working through this section, you will:

  • know how to install R and R studio
  • know how to update R and R studio
  • understand the main set-up of R Studio

2.1 Installing R

Process to install R slightly changes based on the operating system you have.

2.1.1 Installing R for Windows

You can download an installer from here and follow the instructions on your screen to complete the installation

You can also watch this video tutorial if you want visual instructions

2.1.2 Installing R for Mac

Installing R for Mac is similar to Windows. You can download an installer here and follow the instructions or watch this video.

2.1.3 Installing R for Linux distributions

There is no installer to install R for linux distributions such as Ubuntu. Instead we use linux terminal. Open your terminal and type the following command

sudo apt install r-base

Unlike Windows and Mac, there is a comprehensive CRAN package archive packet for linux. You can install it by typing the following command on your terminal

sudo add-apt-repository ppa:c2d4u.team/c2d4u4.0+

2.2 Installing R studio

While R is the programming language and the interpreter, it is very hard to use any programming language directly from the terminal. This is why we have integrated development environments (IDE). IDEs make writing codes, getting results and solving bugs in your program a lot easier.

R can be used with a wide variety of IDEs but the most common and convenient one is R studio. You can download an installer from this link. Installation procedure of Rstudio is the same for Windows, Mac and Linux. So long as you follow the instructions from the installer you should not have any problems

2.3 Updating R and RStudio

Broadly speaking there are two ways to update R on any operation system. The first is to re-installation via the links above and removing the old version. Your Rstudio should still be able to find base R when you run it. Alternatively, you can update R using R with the following code.

if(!require(installr)){
  install.packages("installr")
  require(installr)
  updateR()
}else{
  require(installr)
  updateR()
}

Updating Rstudio is much easier. You can simply click on Help> Check for updates. You can also download an installer for the newer version of Rstudio from the link about and re-install it.
